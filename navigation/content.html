navigation.js 1,360 bytes(minified)

  • navigation() 메서드에 필수 인자로 'hor' 또는 'ver' 키워드를 넣어 메뉴 기능을 실행한다.
  • a 요소 클릭 시 현재 페이지 주소(location.href)와 a 요소의 [href] 값을 비교하여 일치하는 앵커를 강조한다.
  • 현재 페이지 주소와 a[href] 값이 일치하면 a 요소 내부를 strong 으로 감싸고 모든 조상 li 요소에 .active 클래스를 추가한다.
  • a>strong 요소는 현재 페이지와 일치하는 메뉴 이름을 시멘틱하게 강조하는 수단이다. URL 값이 일치하는 앵커만 내부를 strong 요소로 감싼다.
  • li.active 클래스는 현재 페이지 메뉴와 그 상위 메뉴를 스타일링 하는 수단이다.
  • 현재 페이지와 일치하는 앵커 클릭 시 링크 이동을 실행하지 않는다.
